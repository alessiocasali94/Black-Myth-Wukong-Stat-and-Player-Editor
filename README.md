# Black Myth: Wukong Trainer

A trainer application for Black Myth: Wukong built with C++ and Win32 API.

## Build

Requirements:
- Visual Studio 2022 or later
- Windows SDK

Open Project3.vcxproj in Visual Studio, set configuration to **Release / x64** and build.

## Package

Run the packaging script to produce a signed/unsigned release build:

`powershell
.\package.ps1
`
